# Customer Churn Analysis

## Project Overview
This project analyzes telecom customer data to understand churn behaviour using Python, Pandas, and Matplotlib.

## Objectives
- Identify factors influencing customer churn
- Analyze monthly charges and contract types
- Segment customers based on tenure

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Customers with higher monthly charges show higher churn risk.
- Month-to-month contracts have the highest churn.
- New customers (0-1 year tenure) churn more frequently.
- Long-term contracts reduce churn significantly.

## Project Files
- churn_analysis.ipynb — Data analysis notebook
- customer_churn.csv — Dataset
